## Comparison Operators

# Ternary Operator

* A conditional operator that evaluates a condition and returns one of two values based on whether the condition is true or false

<br>

```JavaScript
CONDITION ? EXPRESSION_IF_TRUE : EXPRESSION_IF_FALSE
```
