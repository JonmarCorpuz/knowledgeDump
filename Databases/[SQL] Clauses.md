# SQL Clause Overview

An SQL clause is a SQL statement component that specifies conditions, oganizes, filters, and manages how data is retrieved or manipulated from a database

<br>

<br>

| SQL Clause | Purpose |
| --- | --- |
| `SELECT` | Specifies the column to retrieve from the specified table(s) |
| `FROM` | Specifies the table(s) from where to retrieve the data |
| `WHERE` condition | Filters records based on the specified conditions before grouping or aggregating |
| `GROUP BY` column_name |  |
| `HAVING` value |  |
| `ORDER BY` {ascending|descending} | Sorts the returned rows in the specified order |
| `LIMIT` | Restricts the number of rows returned in the result |
